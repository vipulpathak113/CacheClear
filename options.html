Web Cleaner Extension 1.1

Web Cleaner is an extension for cleaning Cache, Cookies, History,etc on a single click and is made by Vipul Pathak.

How to use :

  • Web Cleaner delete cookies, history, cache etc. Simply click the toolbar icon and Done.
      • However you can change the preference of data to be deleted:
    • Right click on the toolbar and select options
    • Select data to delete and time range.

Privacy :

In order to work properly, Web Cleaner extension requires the permission to access to your history. However, Web Cleaner do not collect nor will ever transmit any of your private data.

How to uninstall :

  1. Click the wrench icon on the browser toolbar.
  2. Click Tools.
  3. Select Extensions.
  4. Find the Web Cleaner extension then click Uninstall.
